# Examples

Standalone, runnable examples. **Not** part of the published Maven reactor —
each example has its own POM that depends on the published openflags artifacts
on Maven Central.

| Example                                        | What it shows                                              |
|------------------------------------------------|------------------------------------------------------------|
| [spring-boot-quickstart](./spring-boot-quickstart) | Minimal Spring Boot 3 app, file provider, targeting rule, MDC, Prometheus. |

More examples will be added as new providers and integrations land. PRs with
new examples are welcome — see [`CONTRIBUTING.md`](../CONTRIBUTING.md).
