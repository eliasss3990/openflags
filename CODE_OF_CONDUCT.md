# Contributor Covenant Code of Conduct

This project adopts version 2.1 of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful, inclusive, and constructive.
- Disagree with ideas, not with people.
- Harassment, discriminatory language, and personal attacks are not tolerated.

## Reporting

Report unacceptable behavior to `elias.gonzalez@ncoders.solutions` with
`[openflags-conduct]` in the subject line. Reports are handled confidentially.
Maintainers may remove comments, commits, code, issues, and contributors that
violate this Code of Conduct.

For the full text and enforcement guidelines, see the link above.
